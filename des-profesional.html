Desarrollo Profesional

Me esfuerzo por implementar las mejores prácticas de desarrollo web, incluyendo pruebas, documentación y optimización para mejorar el rendimiento. También participo de un proyecto en equipo. Se trata de un sitio web para una ONG que rescata caballos, donde he demostrado habilidades como la colaboración y comunicación a traves del método Agile.
Soy una desarrolladora apasionada y comprometida con mejorar mis habilidades. Me gusta estar al día con las últimas tecnologías y herramientas de desarrollo, y siempre estoy buscando oportunidades para aprender y crecer en mi carrera.
Esta página web es un reflejo de mi estilo y enfoque personal. Trabajé para que fuera fácil de navegar y represente mi experiencia, habilidades y pasión por el desarrollo web de una manera clara y concisa. Para lograr esto, utilicé HTML, CSS, Sass y el Framework Bootstrap
Me esforcé por seguir las mejores prácticas de desarrollo, incluyendo la optimización para motores de búsqueda, la accesibilidad, el rendimiento y el diseño responsive. También utilicé Git y GitHub
Estoy emocionada de seguir explorando el mundo del desarrollo web y espero tener la oportunidad de trabajar en proyectos desafiantes e innovadores en el futuro. Gracias por considerar mi página web como parte de mi presentación.
